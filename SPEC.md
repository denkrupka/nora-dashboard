## СТАТУСЫ И АНИМАЦИИ

### WORKING/RUNNING
Идёт к столу, садится, typing (лапы по клавиатуре), код на мониторе (цвет по провайдеру), пьёт из кружки, чешет голову. RUNNING=быстрее + бегает между столами. Иконка задачи над головой, spark-эффект, банка RedBull/кофе/пиво рядом.

### IDLE/DONE/RESERVE
Встаёт, потягивается, идёт через дыру в НОРУ. Рандомная активность каждые ~15 сек:
1. 🍺 Пьёт пиво, BUUUURP
2. 🎮 Играет, высовывает язык, бросает геймпад в стену
3. 🍕 Жрёт пиццу, кусок торчит из рта
4. 😴 Дрыхнет на диване, ZZZ
5. 🧀 Грызёт сыр как настоящая крыса, крошки
6. 🥊 Дерётся с другой крысой (если 2+ idle), мультяшная драка → мирятся с пивом
7. 📺 Смотрит ТВ не моргая, цензурный квадрат
8. 🎸 Дикий танец, другие смотрят
9. 🐱 Стебёт кота-тень лазерной указкой, убегает LOL
10. 💊 Под одеялом, ритмичное движение, "..."→"😅"
11. 🤢 Рыгает в углу, BUUUURP x3, лужа
12. 💩 За коробками, газовое облачко, муха, выходит с газетой
13. 📱 Тупит в телефон, ХА ХА, показывает другим
14. 💤 Спит стоя, качается, падает, встаёт

### SLEEPING
Кровать/диван, одеяло/плед, большие ZZZ, переворачивается, лапа свешивается. При смене статуса — вскакивает, трёт глаза, бежит.

### ERROR
Замирает, голова на клавиатуру (клавиши летят), сползает на пол, X_X, ошибки летят (ERROR 500, NULL POINTER, SEGFAULT), монитор красный. Несколько — кучей в углу.

### DEAD
В норе у стены, мухи, нимб мигает, серый с красным контуром.

## ДОКТОР-КРЫСА NPC
5-10 сек после ERROR: из дыры бежит доктор (халат, красный крест, чемоданчик). Достаёт ОГРОМНЫЙ шприц (3x голова), вкалывает → подскакивает. ИЛИ дефибриллятор → электрошок. Агент воскресает: X→глаза, качается, "💊 Recovering...", прогресс-бар. Доктор хлопает по плечу, убегает. Агент на диван 15 сек → работа.

Для DEAD: пиксельная скорая с мигалкой едет по низу экрана, 2 доктора с носилками, грузят, увозят, агент возвращается сам.

## SQL (для справки)
```sql
CREATE TABLE IF NOT EXISTS agent_status (
  id TEXT PRIMARY KEY, name TEXT NOT NULL, status TEXT DEFAULT 'idle',
  task TEXT DEFAULT '', model TEXT DEFAULT '', provider TEXT DEFAULT 'anthropic',
  tokens_used BIGINT DEFAULT 0, tokens_limit BIGINT DEFAULT 100000,
  commits INT DEFAULT 0, uptime_seconds BIGINT DEFAULT 0,
  last_message TEXT DEFAULT '', updated_at TIMESTAMPTZ DEFAULT NOW()
);
CREATE TABLE IF NOT EXISTS provider_tokens (
  provider TEXT PRIMARY KEY, used BIGINT DEFAULT 0, limit_val BIGINT DEFAULT 1000000,
  unit TEXT DEFAULT 'tokens', updated_at TIMESTAMPTZ DEFAULT NOW()
);
```
